# Interface Gráfica em Java — Projeto acadêmico

Projeto acadêmico para praticar construção de interface gráfica (GUI) em Java e conceitos fundamentais de programação orientada a objetos.

Status: Legacy / projeto acadêmico (não mantido ativamente)

## Objetivo
- Praticar criação de telas e componentes de GUI
- Trabalhar eventos (cliques/ações), campos e validações básicas
- Organizar classes e fluxos simples de interação

## O que este projeto cobre
- Fundamentos de Java (POO, organização do projeto)
- GUI em Java (telas, botões, inputs e eventos — conforme implementação)

## Tecnologias
- Java
- NetBeans / Ant (arquivos nbproject/ e build.xml)
- A pasta dist/ pode conter artefatos de build (ex.: .jar), dependendo do estado do projeto

## Como executar
Opção A — NetBeans (recomendado)
1) Clone o repositório:
   git clone https://github.com/LuisGu06/Interface-Grafica-.git
2) Abra no NetBeans: File > Open Project
3) Execute o projeto: Run

Opção B — Executar .jar (se existir em dist/)
1) Verifique a pasta dist/ e identifique o arquivo .jar
2) Execute:
   java -jar dist/SEU_ARQUIVO.jar
